== Description ==

This plugin is useful for generate rest API url.

Plugin Functionality: 
1) Easy setup from backend
2) User-friendly Settings Interface
3) Generate rest API url from backend and use anywhere.

  
== Get Involved ==

[ViitorCloud](https://viitorcloud.com) believes in an active community support so with such plugins we try to make life easy for developers & of course customers. Don't forget to subscribe our newsletter. 


== Installation == 
 
1) Upload `Easily Generate Rest API Url` to the `/wp-content/plugins/` directory
2) Activate the plugin through the 'Plugins' menu in WordPress
3) Use the Settings->Generate Rest API URL screen to configure the plugin

Plugin Supports:

=> This plugin includes a function to generate rest api url from backend
